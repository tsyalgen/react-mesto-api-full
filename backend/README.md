# Backend часть проекта Mesto

## Установка

`yarn` или `npm install` - устанавливает зависимости

## Запуск проекта

`yarn start` или `npm run start` — запускает сервер   
`yarn dev` или `npm run dev`— запускает сервер с hot-reload


**Используется:**

* JavaScript
* Node.js
* Express.js
* MongoDB
